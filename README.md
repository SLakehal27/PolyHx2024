# PolyHx 2024 - Pollo Vision
Have you ever wondered what bin to throw your trash to? Then this project is for you!

# Disclaimer
**The AI in this project does not come from our own model. We used OpenAI's API to be able to have image recognition features for our project.**

# Installation
```
git clone https://github.com/SLakehal27/PolyHx2024.git
cd PolyHx2024
cd client
npm ci
npm start
```
